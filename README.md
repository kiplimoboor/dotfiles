## Config Files
These config files currently work on _**Arch Linux with Hyprland**_.
It has two themes:
1. Light theme using pastel from [Flexoki](https://github.com/kepano/flexoki)
2. Dark theme using pastel from [Catppuccin](https://github.com/catppuccin)

The theme switch script is mapped to `SUPER + SHIFT + T` in the hyprland config.
